# Description and features
The app includes a semi-transparent, rounded, black background. The sections Polygons, Bridging Blocks, and Pyramids all have input boxes where the user can input any value to match the intended dimensions for the shape they are attempting to create. The settings page is still a work in progress. The new version is expected to release on June. Window control buttons at the top of the sidebar can close and minimize the app. The main purpose of this app is to aid builders in creating complex shapes or bridging points/blocks.
